# !send — Sending Spell Helper

A helper alias for the **Sending** spell and similar magic (sending stones, runestones, etc).  
This command enforces the spell’s 25-word limit while keeping the output clean and flavorful.

---

## Usage

Message text **must be enclosed in quotes**.

```text
!send "message..."
!send -t <target> "message..."
!send -b "message..."
